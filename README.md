# Dota 2 Statistics Bot

Мой первый телеграм бот на Python для анализа статистики Dota 2. Показывает статистику за последние матчи: винрейт, K/D/A, лучших героев, GPM/XPM, основную роль и даже находит самую удачную и неудачную игру.

## Как использовать
1. Отправьте боту свой Steam ID
2. Получите подробную аналитику вашей игры


## Технологии
- Python
- Telebot
- OpenDota API
